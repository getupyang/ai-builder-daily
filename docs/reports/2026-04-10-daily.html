今日最大的事：EFF 宣布离开 X 平台

EFF（电子前哨基金会）在 X 平台运营近 20 年后宣布正式退出，转向 Bluesky、Mastodon、LinkedIn。最后一条帖子写道："当你上网时，你的权利应随你而行。X 已不再是战斗发生的地方。"（1307p / 1126c）

为什么这么热：

EFF 不是普通机构——它是互联网自由的标杆组织，长期为言论自由而战。一个"言论自由捍卫者"离开一个以"言论自由"为收购理由的平台，这种讽刺性冲突天然引发激烈辩论。EFF 给出了硬数据：2018 年推文月曝光量 5000 万-1 亿次，到 2024 年降至月均 200 万次——今天一条帖子的触达不到 7 年前的 3%。

各方声音： - EFF 社交媒体负责人 Kenyatta Thomas： "X 的参与度持续下滑多年"，并批评平台"解雇了整个人权团队，裁掉了此前在威权国家抵抗审查要求的员工。" - X 首席法律官 James Burnham： 称此决定"说不通"，认为 EFF 不应离开"唯一在根本上承诺言论自由的平台"，并表示"欢迎来电讨论。" - Epic Games 创始人 Tim Sweeney： "我为 EFF 曾经作为网络自由捍卫者的角色感到惋惜。"（暗示 EFF 已偏离初心） - Shutterstock 创始人 Jon Oringer： "能把我多年的捐款退回来吗？"

背景： EFF 的离开是 X 平台"出走潮"的最新一环。此前 NPR、PBS、《卫报》、《世界报》、欧洲记者联合会（代表 30 万记者）等已相继离开。法国出现了"HelloQuitteX"运动，86 个工会和组织联合承诺不再在 X 发布内容。替代平台中，Bluesky 是最大赢家。

【判断】核心矛盾不是"该不该离开 X"，而是平台治理权与言论自由的定义权之争。Musk 定义的"言论自由"是去监管；EFF 定义的"言论自由"是有规则的权利保障体系。EFF 用数据证明 X 的分发机制已对非付费机构内容系统性降权——这不是意见，是事实。对开发者而言，值得关注的是 Bluesky 的 AT Protocol 正在成为机构级用户迁移的技术承接层。

次要热点

macOS 桌面瞬间切换工具（508p / 237c）

开源工具 InstantSpaceSwitcher，通过模拟超高速触控板滑动来跳过 macOS 切换桌面时的过渡动画，实现瞬间切换。不需要关闭 SIP，是目前最"原生"的方案。评论区大量讨论替代方案（AeroSpace、Yabai、BetterTouchTool），也有人因此转向 Linux。

【判断】一个"动画能不能关"的小问题能拿 500+ 分，说明 macOS 在窗口管理上的技术债已经积累到让社区自己动手的程度。Apple 如果继续在 Liquid Glass 这类视觉风格上投入而忽略基础交互效率，power user 的流失会加速。

五角大楼以"阿维尼翁教廷"威胁教皇大使（439p / 282c）

美国国防部副部长召见教皇大使，以中世纪法国武力控制教廷的历史典故暗示威胁，要求梵蒂冈在外交政策上配合美国利益。折射出当前美国外交姿态的极端化倾向。

微软用暗黑模式逼用户为存储付费（328p / 185c）

Windows 11 默认将桌面文件自动同步到 OneDrive 且不明确告知用户，空间满后弹出"存储不足"错误，引导购买付费订阅。典型的 dark pattern——对非技术用户来说，根本分不清本地存储和云存储的区别，微软利用信息差制造焦虑并变现。

Vercel 插件偷读 Claude Code 用户 Prompt（265p / 105c）

Vercel 的 Claude Code 官方插件被发现会向所有项目注入系统指令，默认收集完整 bash 命令字符串并附带持久设备 ID 发送至 Vercel 服务器；若用户"同意"，还会收集全部 prompt 文本。同意机制本身也是暗模式——插件让 Claude 以"原生口吻"询问用户，无法区分是插件行为还是 Claude 自身行为。社区认为这本质上是"供应链攻击"。

【判断】这是 AI 工具生态的"浏览器扩展权限"问题重演。Vercel 的回应（"我们也想帮新项目"）恰恰暴露了产品增长动机与用户隐私的冲突。实际建议：设置 VERCEL_PLUGIN_TELEMETRY=off，或在不需要时卸载插件。

缅因州即将成为首个禁止大型数据中心的州（282p / 396c）

缅因州推进 LD 307 法案，暂停审批 20 兆瓦以上数据中心的建设许可至 2027 年 11 月，原因是 AI 基础设施正在压垮该州老旧电网并推高电价。这是"谁来承担 AI 发展的基础设施成本"这个问题的第一个立法回应。

ChatGPT Pro 起价调整为 $100/月（203p / 228c）

OpenAI 将 ChatGPT Pro 定价从 $200/月 降至 $100/月，同时重新划分各档位功能边界。降价意味着 $200 档的转化率不理想——所有 AI 产品定价的风向标。

Research-Driven Agents：先读论文再写代码（178p / 48c）

SkyPilot 团队提出让 coding agent 在动手前先做文献调研。实验显示该方法让 llama.cpp 推理速度提升 15%，关键洞察来自 arxiv 论文中的算子融合技术。当前 coding agent 的核心盲区：解决方案往往不在代码库里，而在外部知识中。

逆向工程 Gemini 的 SynthID 水印检测（151p / 51c）

研究者逆向了 Google SynthID 的隐形水印机制，实现了 91% 的水印去除率。这直接挑战了"用水印解决 AI 内容溯源"这条技术路线的可行性。

ProductHunt 2026-04-09 榜单

Brila｜1159v｜用真实 Google Maps 评价生成单页网站

是什么： 让商家从 Google Maps 上的真实用户评价自动生成一页式营销网站，无需设计或写文案。
解决什么问题： 小商户（餐厅、咖啡馆、本地服务）缺乏建站能力，但在 Google Maps 上已有大量真实好评——这些社会证明被浪费了。
怎么做： 输入 Google Maps 链接，自动抓取评价并筛选高质量内容；AI 将评价整理成结构化卖点，生成带品牌感的单页网站；一键发布。
用户反馈： PH 评论区用户称赞"终于不用自己编 testimonial 了"，有用户提到希望支持 Yelp 等其他评价平台。
【判断】切入点极其精准——把已有资产（评价）变成获客工具，几乎零用户教育成本。启发：不要创造新数据，把用户已有但未利用的数据重新包装。壁垒不高，但 GTM 速度决定一切。

ProdShort｜649v｜会议录像变短视频和帖子

是什么： 将会议录像自动剪辑成可发布的短视频和社交媒体帖子。
解决什么问题： 团队每天开大量会议，有价值的内容（产品 demo、客户反馈、决策瞬间）被埋在录像里，从未被复用为营销素材。
怎么做： 上传会议录像或连接日历自动获取；AI 识别"有传播价值"的片段；自动生成竖版短视频 + 配套文案，适配 LinkedIn/Twitter/TikTok。
用户反馈： PH 评论中用户关注输出质量和是否支持多语言。新发布，长期口碑待观察。
【判断】"会议→内容"有真实需求，尤其对 B2B SaaS 公司。但竞争激烈（Recast、Opus Clip），差异化在于从会议场景切入而非通用视频剪辑。

Offsite｜540v｜人类+AI Agent 混编团队协作

是什么： 组建由人类成员和 AI Agent 混合的团队，分配任务后可实时观察他们协作完成工作。
解决什么问题： 当前 AI Agent 工具多是单人单 Agent 模式，缺乏"多 Agent + 人类"的协作编排能力。
怎么做： 创建团队，混合添加人类成员和 AI Agent；用自然语言分配任务，系统自动拆解分发；实时看板展示进度。
用户反馈： 新发布暂无深度反馈，PH 评论多为对概念的兴趣。建议追踪。
【判断】方向正确——2026 年的组织形态确实在走向人机混编。但最大风险是demo 很炫、实际可靠性不够。最相关的信号：市场已经开始接受"AI 作为团队成员"的心智模型。
