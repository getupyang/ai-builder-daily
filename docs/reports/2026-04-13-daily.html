今日最大的事：Docker 在西班牙被足球联赛"误杀"

HN 709分 / 272评论，4/12 当日最高。

西班牙足球甲级联赛（La Liga）为打击盗版直播，获法院授权可在比赛日随时封锁任意 IP 地址。问题是他们封的是 Cloudflare 的 IP 段——而 Docker Hub、GitHub、Google Fonts、甚至政府网站都跑在 Cloudflare 上。结果：每到比赛日，整个西班牙的 docker pull 就挂掉。

这不是个例。自 2025 年 2 月 La Liga 首次启动 IP 封锁以来，受影响的合法服务清单越来越长。Cloudflare 已向西班牙高等法院提起上诉，但 2025 年 3 月一审败诉。开发者社区的愤怒点在于：一个体育联盟的版权诉求，正在制造事实上的互联网审查——而且是 IP 级别的粗暴封锁，不是精准的域名拦截。

【判断】 这件事的深层问题是：当 CDN（Cloudflare、Fastly）成为互联网基础设施，IP 级别的封锁就变成了"一刀切断整条街来抓一个小偷"。这个模式一旦被其他国家效仿，后果远超西班牙一国。对开发者的实际影响：如果你的用户在西班牙，考虑多 CDN 备份。

今日第二大事：Claude Pro Max 5x 用户 1.5 小时就用完配额

HN 552分 / 505评论。

一位 Claude Pro Max 5x 用户（$100/月）报告：正常使用（问答+轻度开发），配额 1.5 小时就耗尽。505 条评论说明这不是个案——大量用户遇到同样问题。

根因是多重叠加的： 1. 缓存 TTL 静默降级：3 月初 Anthropic 将 prompt cache TTL 从 1 小时降到 5 分钟（同期另一条 HN 496分/383评论），cache_read token 仍按全价计入配额 2. 1M 上下文窗口的代价：每次 API 调用发送 100k-960k token，工具密集型的 Claude Code 使用模式下每小时 200+ 次调用 3. 后台 session 吸血：其他终端里挂着的 session 持续消耗配额（compact、retro、hook 处理） 4. 高峰期加速消耗：太平洋时间 5am-11am 配额消耗更快

Anthropic 已承认问题，称"这是团队最高优先级"。

【判断】 这是 Anthropic 商业模式的结构性矛盾：Claude Code 的价值在于"无限使用"的心智模型，但 1M 上下文 + 工具密集型 agent 的 token 消耗远超传统聊天。如果不解决定价/配额的透明度问题，Pro Max 用户会加速流向本地模型（上周的 Gemma 4 热潮不是巧合）。对你来说：这也解释了为什么你的 Claude Code 用量有时会突然耗尽。

七国实现 100% 可再生能源发电

HN 530分 / 276评论。

阿尔巴尼亚、不丹、尼泊尔、巴拉圭、冰岛、埃塞俄比亚和刚果民主共和国——这七个国家已实现 100% 可再生能源发电。核心驱动力都是水电（冰岛加上地热）。巴拉圭 90% 电力来自与巴西共有的伊泰普大坝。

HN 评论区的核心争议：这些都是人口小国或能源消耗极低的国家，不能简单类比到工业化大国。但作为"可再生能源能否独立支撑一个国家电网"的证明，这些数据是有说服力的。

Google 下架《心跳文学部》引发内容审核争议

HN 344分 / 167评论。

Google 以"描绘敏感主题"为由将经典心理恐怖视觉小说《Doki Doki Literature Club》从 Play Store 下架——但这款游戏已经在 Play Store 上架了 4 个月（2025年12月上架），且在 iOS 上仍正常销售。开发者 Dan Salvato 和发行商 Serenity Forge 正在申诉。

【判断】 平台审核的不一致性是这件事的核心：同一款游戏，上架时不违规，4 个月后突然违规？iOS 不违规，Android 就违规？这种不透明的审核标准对独立游戏开发者的伤害是实实在在的。

ProductHunt 2026-04-12 榜单

Gemini Interactive Simulations｜327v（日冠）

Google 给 Gemini 加了交互式 3D 模拟功能。你问一个物理概念，它不只是文字解释——直接在对话框里渲染一个可操作的 3D 模型。比如双缝实验，你可以拖动滑块调节参数，实时看干涉条纹变化。

技术上这是 Google 的强项（WebGL + Gemini Pro 模型），但产品意义更大：这是第一次有 AI 聊天界面在"输出格式"上做出实质性突破。文字、图片、代码之外，"交互式模拟"成为新的输出模态。需要在 gemini.google.com 上用 Pro 模型才能触发。

【判断】 这对教育场景的冲击最大。传统的 Khan Academy 式视频教学和 Gemini 式交互模拟之间的体验差距，可能比大多数人预想的更大。

瞄准的是"懒得打开银行 App 但整天在终端里"的开发者群体。切口很窄，但 product-market fit 可能很精准——229 票说明开发者真有这个需求。

ElevenLabs Music Marketplace｜226v

ElevenLabs 推出 AI 音乐交易市场。用它的 AI 生成音乐后，可以直接发布到市场，其他人付费使用/remix 时你赚钱。创作者分成从 25% 起。三档授权（社交媒体/付费营销/线下使用）。

背景数据：自 2025 年 8 月推出 Eleven Music 以来，已生成 1400 万首歌。Voice Marketplace 已累计向创作者支付 1100 万美元。

【判断】 ElevenLabs 正在从"AI 语音工具"转型为"AI 音频平台"——Voice Marketplace → Music Marketplace 的逻辑是清晰的。但 25% 的创作者分成在音乐行业偏低（Spotify 也被骂分成低），这会是长期争议点。
